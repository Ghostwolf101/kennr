## Body text

Most design inspection tools give you raw CSS dumps. Kennr is different — it captures the outerHTML and a screenshot of whatever page you're on, runs it through an AI layer, and gives you an interpreted design summary: color roles, type hierarchy, spacing logic. Patterns, not just values.

Built it because I kept reverse-engineering sites by hand. One click is faster.

**How to install (sideload — no store required):**
1. Download the zip from the release page and unzip it
2. Chrome: `chrome://extensions` → Developer mode → Load unpacked
3. Edge: `edge://extensions` → Developer mode → Load unpacked
4. Point the ᚲ icon at any page and extract

Apache 2.0. Backend is a Cloudflare Worker — self-hostable or use the default endpoint. Privacy policy at web.wyerd.org/kennr/privacy.
